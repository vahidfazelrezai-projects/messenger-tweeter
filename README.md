# messenger-tweeter
post tweets from Messenger

For this to work, include a file called `creds.js` containing the following credentials:
```javascript

module.exports = {
    'FACEBOOK_CRED': {
        email: '...',
        password: '...'
    },
    'TWITTER_CRED': {
        consumer_key: '...',
        consumer_secret: '...',
        access_token_key: '...',
        access_token_secret: '...'
    }
}
```
